## 🎯 What is Source Code DarkLight?

**Source Code DarkLight** is an advanced web security analysis tool that combines traditional pattern-based vulnerability detection with **AI-powered analysis**. It's designed for security professionals, developers, and bug bounty hunters to identify and understand security flaws.

Unlike traditional scanners, DarkLight **teaches** you why vulnerabilities are dangerous and how to fix them.

## ✨ Key Features

### 🔍 Deep Source Code Analysis
- Scrapes HTML, JavaScript, CSS, and inline event handlers
- Extracts forms, scripts, and comments
- Follows internal links (crawling mode)

### 🤖 AI-Powered Insights
- Uses Ollama (local AI) for analysis
- Provides explanations in simple terms
- Shows real-world impact scenarios
- Gives step-by-step remediation with code examples

### 📊 Multi-Format Reports
- **HTML**: Interactive web report with expandable sections
- **JSON**: Machine-readable format for integration
- **Markdown**: Clean text-based documentation

### 🕷️ Web Crawling
- Can scan entire websites
- Configurable page limits
- Same-domain crawling

### 🎨 Beautiful CLI
- Color-coded output
- Progress indicators
- Real-time status updates
